# IT-Infrastructure-Project-NOVATECH-SOLUTION-Pvt-Ltd
**NovaTech Solutions Pvt Ltd**

## Project Description
This project demonstrates the design and implementation of a small company IT infrastructure with a mini data center. The infrastructure provides a secure, reliable, and scalable network environment for organizational operations.

The network was designed using a Star Topology in Cisco Packet Tracer. The virtual infrastructure consists of one Windows Server and two Windows Client virtual machines. The Windows Server is configured to provide essential network services, while the client machines are connected to the server for communication and resource sharing.

The project also includes web server configuration, allowing users to access the company website hosted on the Windows Server through client computers. This implementation demonstrates server-client communication, network connectivity, and basic enterprise IT infrastructure deployment.
